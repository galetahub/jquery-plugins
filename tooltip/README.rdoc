= Simple tooltip plugin for jquery

Show tooltip on bottom of elements.

== Usage

  <a href="#" title="Some tooltip">...</a>
  
  <script>
    $("a").tooltip();
  </script>

== Settings

* activation: "hover"
* keepAlive: false
* edgeOffset: 3
* position: "bottom"
* delay: 400
* fadeIn: 200
* fadeOut: 200
* attribute: "title"
* content: false
* container_id: "tt-container"
